# Arogyanvesha — Where AI Meets Ayurvedic Healing

> आरोग्यान्वेषा — The Quest for True Wellbeing

A comprehensive AI-powered Ayurvedic healthcare platform bridging ancient Vedic wisdom with modern artificial intelligence.

---

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm or yarn

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
